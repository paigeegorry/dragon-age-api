Routes + Queries To Get
GET /api/v1/characters all characters
(default 20 per page / 792 total)
GET /api/v1/characters?page=${number} all characters:
pagination
GET /api/v1/characters?perPage=${number} all characters:
customize how many show up per page
GET /api/v1/characters/${CharacterId} a character by their id
GET /api/v1/characters/random one random character
GET /api/v1/characters?appearances=${value} characters by appearance
which game/comic they appeared in
GET /api/v1/characters?quests=${value} characters by quest
